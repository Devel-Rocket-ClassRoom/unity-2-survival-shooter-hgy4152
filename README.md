# Survival Shooter 모작 과제

## 문제

C#과 유니티 엔진을 사용하여 윈도우 플랫폼에서 제공된 탑다운 시점 3D 슈터 게임 **Survival Shooter**를 모작하는 개인 과제다. 완성본 게임을 플레이하며 분석하고, 제공된 애셋을 활용하여 핵심 메커니즘과 기능을 동일하게 구현해야 한다.

## 내용

1. 게임 분석: 완성본 게임을 플레이하며 핵심 메커니즘, UI/UX, 레벨 디자인 등을 파악한다.
2. 개발 환경 설정: C#과 유니티 엔진을 사용하여 개발 환경을 구성한다.
3. 필수 기능 구현: 게임의 핵심 기능을 다음과 같이 구현한다.
    - 캐릭터 이동 및 조작 (이동, 회전 등)
    - 기본 공격 (사격)
    - 적 AI (등장, 추적, 공격, 피격, 사망 등)
    - 적 스폰 (복수 스폰 포인트, 난이도 변화 등)
    - UI (체력, 점수, 게임오버 등)
    - 기타 게임 로직 (카메라, 레벨, 충돌 처리 등)
4. 자유 구현(선택): 필수 기능 구현 이후 자유롭게 기능을 추가하거나 변경할 수 있다.
    - 새로운 공격 형태 (무기 교체, 특수 공격 등)
    - 외부 리소스 활용 (모델, 텍스처, 사운드 등)
    - 기타 (아이템 드롭, 웨이브 UI, 미니맵 등)
5. 테스트 및 디버깅: 구현된 게임을 충분히 테스트하고, 발견된 버그와 문제점을 수정한다.
